# Análise de Dados de Aluguéis de Imóveis

## Sobre o projeto
O projeto tem como objetivo realizar uma análise exploratória de dados (EDA) em uma base de imóveis para aluguel. Durante o processo foram realizadas etapas de limpeza, tratamento, filtragem e manipulação dos dados para extrair informações sobre os imóveis.

A análise busca entender características como:
- Tipos de imóveis
- Valores médios de aluguel
- Distribuição dos tipos de propriedades
- Filtros de imóveis baseando-se em especificações

## Objetivos da análise

- Explorar e entender a estrutura da base de dados
- Identificar o valor médio de aluguel por tipo de imóvel
- Filtrar apenas os imóveis residenciais relevantes
- Remover dados inconsistentes ou nulos
- Criar novas colunas para facilitar a análise
- Gerar subconjuntos de dados com critérios específicos

## Dataset

O dataset utilizado contém informações sobre imóveis disponíveis para aluguel, incluindo características das propriedades e valores associados.

Principais atributos presentes:

- Tipo do imóvel
- Bairro
- Área do imóvel
- Número de quartos
- Número de vagas de garagem
- Valor do aluguel
- Condomínio
- IPTU

Os dados foram utilizados para fins de estudo e prática de análise de dados.


## Tecnologias utilizadas

- Python
- Pandas
- Jupyter Notebook

## Principais insights

- Apartamentos representam a maior parte dos imóveis disponíveis para aluguel na base de dados.
- Imóveis com maior número de quartos tendem a apresentar valores de aluguel significativamente mais altos.
- A maioria dos apartamentos disponíveis possui entre 1 e 2 quartos.
- Ao aplicar filtros de preço e área, é possível identificar imóveis com melhor custo-benefício dentro da base.
